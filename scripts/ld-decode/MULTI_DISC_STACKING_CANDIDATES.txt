RECOMMENDATIONS FOR STACKING TESTS
===================================

Best candidates (in order):
1. Domesday NationalB_PP (PAL CLV) - 7 copies, large dataset
2. The World the UN and You (PAL CAV) - 7 copies, excellent for comprehensive testing
3. Dragon's Lair (NTSC CAV) - 4 usable copies, arcade game
4. British Garden Birds (PAL CAV) - 4 copies each side

Key considerations:
- All listed sets have identical file position ranges across copies
- VBI ranges vary due to different starting positions on physical discs
- File position alignment is what matters for stacking
- Larger datasets (50k+ frames) provide more test coverage
- Mix of CAV and CLV formats provides format diversity
- Mix of NTSC and PAL provides standard diversity
