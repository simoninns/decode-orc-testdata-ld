TBC File Size Limits
====================

Based on actual .tbc file sizes from ld-decode:

NTSC:
  - ~935 KB per frame (0.935 MB)
  - 50 MB limit = ~53 frames maximum

PAL:
  - ~1.4 MB per frame
  - 50 MB limit = ~35 frames maximum

These limits apply when decoding frames with ld-decode --length parameter.
